Description

The Railway Management System is a console-based application developed using C++ and Object-Oriented Programming (OOP) concepts. It allows users to manage train records by adding new trains, displaying all trains, searching trains by train number, and tracking the number of train objects using a static member.

Features
Add new train records
Display all train records
Search train by train number
Count total train objects
Menu-driven interface
Uses OOP concepts such as Encapsulation, Constructors, Destructors, and Static Members
